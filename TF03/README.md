# TF03

## Iniciando

- Clone repo

```bash
    git clone https://github.com/Diego0Alves/TF-Implementacao-de-software.git
```

- Entre ne diretório do repo local

```bash
    cd TF-Implementacao-de-software/TF03
```

- Crie o .env

```bash
    copy .env.example .env
```

- Inicie com Docker

```bash
    docker compose up --build
```
